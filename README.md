# GRID07 AI ASSIGNMENT

## Overview
This project simulates a simple AI system with three main components:
- Routing posts to relevant bots
- Generating structured content
- Defendng responses in a conversation

## Phase 1: Routing
Posts are analyzed using keyword-based logic to match relevant bot personas.

## Phase 2: Content generation
Each bot generates a structured JSON response based on its personality.

## Phase 3: Defense mechanism
The system detects prompt injection attempts and maintains consistent behavior.
